# 00. Meta Context

Purpose:
- Modular context system for AI assistants in the `loan-bre-system` project.
- Replaces the single-file monolithic context approach.

Operating rule:
- Update this context system at the end of each work session.

Scope:
- This system covers developer profile, architecture, development rules, current status, and domain glossary.

Language policy:
- Internal context and AI instruction files must be written in English.
- The project root `README.md` remains in Spanish by project convention.
